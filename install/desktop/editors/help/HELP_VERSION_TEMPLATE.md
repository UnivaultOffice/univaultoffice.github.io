Help version template (manual update)

Use this checklist when adding a new help bundle version.

1) Create a new folder:
   install/desktop/editors/help/vX.Y.Z/apps

2) Copy the placeholder bundle from v1.0.0/apps:
   - index.html
   - style.css
   - word.html
   - cell.html
   - slide.html
   - pdf.html
   - forms.html
   - diagram.html

3) Update the title/footer to the new version:
   - "vX.Y.Z"

4) Replace placeholder content with real help pages when ready.

5) Ensure URL_WEBAPPS_HELP points to:
   https://univaultoffice.github.io/install/desktop/editors/help/vX.Y.Z/apps
